Map shape ⚡ feeds the motor files

      

      

      

      

      

      
The map is stored normalized (% of each motor's max rpm
      and peak torque), so one shape fits any motor you apply it to. Grid is
      the model's native 15×14.

Applying pins this map to the motor (visible in its
      Vehicle Builder card); every run's generated .mat then uses it. The
      CSV uses that motor's physical rpm/N·m breakpoints — the same format
      the upload button accepts.
